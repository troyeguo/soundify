### Develop

Apply Spotify Client ID [HERE](https://developer.spotify.com/) or use my Client ID in the .env file.

```bash
$ git clone https://github.com/troyeguo/soundify.git
$ cd soundify
$ yarn
$ yarn dev
```

Enjoy!
